Guide du Terminal

Commandes de base

  • help - Affiche la liste des commandes
  • ls - Liste les fichiers disponibles
  • cat [fichier] - Affiche le contenu d'un fichier
  • clear - Efface l'écran
  • open [fichier] - Ouvre un fichier dans une fenêtre séparée

Fonctionnalités

  • Utilisez Tab pour l'autocomplétion
  • Utilisez ↑ et ↓ pour naviguer dans l'historique
  • Les fichiers images peuvent être ouverts avec la commande open

Easter Eggs

Il existe des commandes cachées à découvrir... Essayez des commandes comme matrix, hack ou snake !
